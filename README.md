# Vivek Kumar - Personal Portfolio

Welcome to the source code of my personal portfolio website! 🚀

This repository contains the front-end codebase for my professional portfolio, blending a retro 8-bit aesthetic with modern, high-performance SaaS landing page UI components. 

## 🌟 Overview

I'm Vivek Kumar — an Engineering student, Quant researcher, and Developer. This portfolio is designed to showcase my projects, skills, and technical background in an interactive and visually striking format.

### Key Features
- **Dynamic Hero Section:** A beautiful pixel-art background with typing animations and interactive status chips.
- **Interactive Mindmap:** A custom SVG-powered animated node graph that maps out my core competencies across Engineering, AI, Quant Research, Digital Media, Systems Design, and Data Analytics. Hovering over nodes updates an adjacent retro terminal console with simulated logs.
- **Feature Triad:** A modern card-based layout highlighting my core philosophy: *Build First, Research & Iterate, Create With Purpose*.
- **Technical Arsenal:** A sleek, categorized grid displaying my favorite tools, languages, and frameworks.
- **Clean Responsive Design:** Built from scratch with Vanilla HTML, CSS, and JavaScript. Fully responsive for mobile and desktop screens.

## 🛠️ Tech Stack
- **HTML5** - Semantic structure and layout
- **CSS3** - Custom styling, flexbox/grid layouts, CSS variables, and keyframe animations
- **Vanilla JavaScript** - DOM manipulation and interactive SVG logic

## 🚀 Getting Started

To run this project locally, simply clone the repository and open `index.html` in your favorite web browser. No build steps or complex dependencies required!

```bash
git clone https://github.com/ehvivek/Portfolio.git
cd Portfolio
open index.html
```
